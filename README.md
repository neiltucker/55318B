# 55318B Advanced Automated Administration with Windows PowerShell (Updated)
Courseware Access:  https://www.softwaretutorialservices.com/course/55318b

55318B is the updated version of 55318A. This three-day course teaches experienced IT professionals how to build reliable, reusable, and production-ready automation with modern PowerShell. The course updates the previous Windows PowerShell-focused curriculum for current corporate environments that use PowerShell 7.x, Windows Server 2025, SQL Server 2025, RESTAPIs, Git-based workflows, and secure automation. Students create advanced functions, orchestrate complex workflows, manage errors securely, and integrate AI-assisted analysis tools.


Audience:<br>
This course is intended for IT professionals who are experienced in general Windows Server and Windows Client administration. Students should have a foundational knowledge of Windows PowerShell.  In addition, this course provides scripting guidance for Microsoft Azure administrators and developers who support development environments and deployment processes.

After completing this course, students will be able to:<br>
• Design and deploy production-grade advanced functions, script modules, and controller scripts using modern PowerShell 7.6 <br>
• Integrate external interfaces, call REST APIs, parse JSON payloads, and retrieve datasets from SQL Server 2025 databases safely <br>
• Implement enterprise exception trapping, system event logging, task scheduler execution, and multi-threaded background jobs <br>
• Establish robust security controls, protect administrative secrets, build JEA endpoints, and structure projects using Git version control <br>
• Utilize modern testing, static debugging tools, and AI validation methods to inspect automated code structures safely <br>

The material has been updated to teach the use of newer PowerShell tools and how to incorporate AI into PowerShell workflows.
