*****   Class Delivery   ***** 
55318B is the updated version of the 55318A course.  
This three-day course teaches experienced IT professionals how to build reliable, reusable, and production-ready automation with modern PowerShell. The course updates the previous Windows PowerShell-focused curriculum for current corporate environments that use PowerShell 7.x, Windows Server 2025, SQL Server 2025, RESTAPIs, Git-based workflows, and secure automation. Students create advanced functions, orchestrate complex workflows, manage errors securely, and integrate AI-assisted analysis tools.
